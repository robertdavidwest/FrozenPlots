# FrozenPlots
FrozenFlask combined with Flasl-Flatpages (and to featuring Plotly DataVis)

Goal: be able to produce pretty static reports that can be sent to clients the include beautful and interactive visulisations 


I used this guide: https://nicolas.perriault.net/code/2012/dead-easy-yet-powerful-static-website-generator-with-flask/


$ pip install Flask Frozen-Flask Flask-FlatPages
